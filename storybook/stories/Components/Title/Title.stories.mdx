# Title Documentation

Title can be used to create headings for a page or each major page sections. It also defines the different styles for heading.

<br/>
<br/>

## COLORS

Title component supports all colors available in the design system which can be applied via Styles Dropdown

``
  classname pattern: text--<color-token>
``

<br/>

<Preview>
  <Story name="color" id="color">
    <div class="link-list">
      {
        TitleProps.STYLE_SYSTEM_PROPS.COLORS.map(colorEl=> (
          <div className={colorEl}>
            <title-web-cmp 
                text={colorEl.replace('-',' ')}
